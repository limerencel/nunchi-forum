# 产品需求文档 (PRD)

## 项目概述

**项目名称**: Nunchi Forum
**定位**: 现代简约的论坛系统
**目标用户**: 技术社区、产品团队、开发者

## 核心功能 (MVP)

### 1. 用户系统
- 用户注册/登录
- 个人资料管理
- 用户等级/权限

### 2. 论坛结构
- 板块（Forum）管理
- 主题（Thread）发布
- 回复（Post）功能
- 标签（Tag）系统

### 3. 互动功能
- 点赞/踩
- 收藏
- @提及
- 通知中心

### 4. 内容管理
- 富文本编辑
- 代码高亮
- 图片上传
- 搜索功能

### 5. 治理功能
- 举报
- 审核
- 禁言

## 页面结构

1. **首页** - 板块列表、热帖、最新帖
2. **板块页** - 主题列表、筛选排序
3. **主题详情页** - 帖子内容、回复列表
4. **发布页** - 创建新主题
5. **用户中心** - 个人资料、我的帖子、设置
6. **搜索页** - 全站搜索

## 设计风格

- **风格**: 现代简约
- **主题**: 支持暗黑/亮色模式
- **响应式**: 移动端优先

## 技术栈

- 前端: React 18 + Vite + TypeScript + Tailwind CSS
- 后端: Node.js + Fastify + TypeScript
- 数据库: PostgreSQL
- 部署: Docker Compose

## 版本规划

### V1.0 MVP (4周)
- 用户注册/登录
- 板块/主题/回复 CRUD
- 基础互动（点赞、收藏）
- 通知系统

### V1.1 (2周)
- 富文本编辑器
- 图片上传
- 搜索功能
- 管理后台

### V2.0 (4周)
- 标签系统
- 用户等级
- 主题推荐
- 实时通知
