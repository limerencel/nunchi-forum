# Nunchi Forum

一个现代简约的论坛系统。

## 技术栈

- **前端**: React 18 + Vite + TypeScript + Tailwind CSS
- **后端**: Node.js + Fastify + TypeScript
- **数据库**: PostgreSQL
- **部署**: Docker Compose

## 项目结构

```
nunchi-forum/
├── frontend/          # React 前端
├── backend/           # Fastify 后端
├── docs/              # 文档
├── tests/             # 测试
└── docker-compose.yml # 部署配置
```

## 快速开始

```bash
# 克隆项目
git clone https://github.com/limerencel/nunchi-forum.git
cd nunchi-forum

# 启动所有服务
docker-compose up -d

# 访问
前端: http://localhost:3000
后端: http://localhost:3001
```
